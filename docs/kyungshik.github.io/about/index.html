About

넘어짐의 연속이었지만, 철저한 전략으로 다시 일어납니다. 20대 초반에 작은 IT 기업에서 근무했습니다. 이론과 사례분석으로는 경영을 이해할 수 없다고 생각했습니다.

벤처기업에서 성장 함께 성장하며 실패와 극복하는 방법을 배웠습니다. 기업 공개 TF 팀의 경험과 국가사업의 성과 통계 분석 경험을 통해 경영 전략 능력을 키웠습니다. 해외 투자를 위해 대표님과 4명으로 구성된 팀에 속해 미국, 중국, 인도네시아, 베트남에서 사업설명회를 진행했습니다. 회사가 나아가고자 하는 방향을 직접적으로 이해할 수 있었습니다. 가장 기억에 남는 활동은 미국 마이크로소프트 Inspire 2019 파트너 영업사로 참여한 경험입니다.

열정에 앞서 실패할 때가 많습니다. 그러나 실패 요인을 분석하고 새로운 전략을 수립한다면 더 나은 성과를 이룰 수 있다는 교훈을 얻었습니다.
